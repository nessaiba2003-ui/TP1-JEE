### TP 1 : Programmation dynamique par réflexion

### Objectif
Ce TP vise à comprendre l'utilisation de la programmation dynamique en Java afin de:
aboutir a lInstanciation des classes dynamiquement et Injecter des dépendances via la réflexion puis Invoker des méthodes dynamiquement
 et reussir a Charger des classes sans configuration statique et finally finish la creation une application configurable dynamiquement
### Prérequis
Java 8 ou supérieur
Connaissance de la Programmation Orientée Objet
Notions de base en Java

### Avantages de la Programmation Dynamique

Flexibilité que ce voit en Changeant d'implémentation sans recompiler
Configuration externe avec la  modification le comportement via fichiers
Injection de dépendances avec le principe des frameworks comme Spring
Plugin architecture en Chargeant des modules dynamiquement
Testing qui facilite les tests unitaires 
⚠️ Inconvénients
 Plus lent que l'appel direct
 Erreurs détectées à l'exécution, pas à la compilation
 Code plus complexe à comprendre
